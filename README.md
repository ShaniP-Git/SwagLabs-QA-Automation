# Swag Labs (SauceDemo) - End-to-End QA Project

## Project Overview
This project demonstrates a comprehensive QA process for the **Swag Labs** e-commerce website. 
It covers the entire STLC (Software Testing Life Cycle), from initial planning to automated execution.

## Project Structure
* **Documents/** - STP, STD, STR, and Bug Report (PDF).
* **Pages/** - Page Object Model (POM) classes.
* **Tests/** - Automated test scripts (Pytest & Playwright).
* **Reports/** - Test execution reports and screenshots.
